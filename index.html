Instants is a full-stack photo application inspired by Instagram. The app's API is built using Ruby on Rails, with the frontend composed of React components using Redux's uni-directional flux pattern.

DynamicBase

DynamicBase is a lightweight object-relational mapping tool built with Ruby. By creating a subclass of the Base class, tables in the database are mapped to new class instances providing all the benefits of OOP that we know and love.

Repo

WordCube

WordCube is a fun word game built with vanilla JavaScript. The game uses a binary search function to validate words from a 70,000 word dictionary and scores words based on length.
